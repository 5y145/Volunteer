# 📝 Introduce
```
1365 자원봉사 사이트의 정보를 앱에서 쉽게 조회할 수 있습니다.
 1. 지역별로 조회
 2. 세부 검색으로 조회
 3. 봉사정보 세부사항 조회
 4. 신청 화면으로 이동
 5. 북마크 기능
 
 공공데이터 포털의 Open Api를 사용하였습니다.
 - 행정안전부 봉사참여정보 서비스
 - 행정안전부 코드조회 서비스
```
