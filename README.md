## Introduce

```
1365 봉사활동 사이트의 정보를 쉽게 조회할 수 있습니다.
 - 지역별로 조회
 - 세부 옵션으로 조회
 - 세부사항 조회
 - 신청 웹 페이지로 이동
 - 북마크 기능

공공데이터 포털의 Open Api를 사용하였습니다.
 - 행정안전부 봉사참여정보서비스
 - 행정안전부 코드조회서비스
```
